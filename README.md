# My Calculator

My Calculator! Here you can perform your calculations in a hilariously peculiar UI, made by me.

You can do multiple things, like

- Basic arithmetic operations: addition, subtraction, multiplication, division, modulo
- Handle decimals with precision up to 6 places
- Clear the display or delete the last character
- Toggle whether to clear after showing a result
- Press the Boop button for a fun alert
- Error handling for invalid operations like division by zero

And it's funny because of the peculiar design with bright colors, dashed borders, and quirky elements that make it stand out from boring calculators!
